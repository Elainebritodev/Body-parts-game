# Game: Quiz "Body parts for kids"
![alt text](https://github.com/ramunizbh/project1/blob/master/src/images/river2.png?raw=true)

### O QUE É 
Um jogo para computador que pode ser utilizado como ferramenta no ensino da língua inglesa.


### A QUEM SE DESTINA - OBJETIVO 
Este projeto é mantido sob o domínio público. O objetivo deste documento é facilitar a compreenção do escopo do projeto e de como funciona o jogo.


### INSTALAÇÃO LOCAL / LOCAL INSTALLATION 

• Em qualquer navegador do seu computador acesse o link: https://ramunizbh.github.io/project1/


### Regras do Jogo 
Convenções adotadas no game para ganhar:
* Para começar o jogo escolha opção **Sim**, em seguida clicar no botão "OK" e "Próximo"
* Identifique a parte do corpo que está apontada e sem preenchimento e escolha a opção correspondente
* Para confirmar a escolha clique no botão "verificar"
* Para passar para o próximo desafio, clique no botao "Próximo"
* **Para Ganhar:**
* *É preciso identificar de forma correta todas as partes do corpo da menina de forma correta



### BIBLIOTECAS DO PROJETO 

* Jogo feito usando HTML, CSS e JavaScript
